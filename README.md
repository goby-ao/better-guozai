## 这是什么

“果仔的一天”是一款为孩子和家长共同设计的日常成长 App。孩子可以查看今天要做什么，用点击或手指划线完成任务，记录阅读、运动等实际完成量，并留下当天的心情与回顾。家长负责维护长期计划、补充鼓励、回看趋势和管理数据。

它不把习惯培养变成另一张成绩单。这里没有金币余额、扣分、排行榜，也不会因为中断一天就把努力清零。我们更在意孩子有没有开始、是否愿意继续，以及那些小小的进步能不能被看见。

## 核心体验

### 🌤 清楚地过好今天

- 必做任务与选做任务分开呈现，孩子一眼就能看懂今天的计划。
- 支持点击复选框和手指横向划掉两种打卡方式。
- 阅读、运动、练习等任务可以同时记录目标与实际完成量。
- 孩子可以主动添加“我的挑战”，把自己的想法放进当天计划。
- 每完成一项任务，都会出现与任务内容相关的过程型夸奖和轻快音效。

### 🌱 把坚持种成成长花园

完成当天全部必做任务，就会点亮一次“今日达成”，花园里的植物也会成长一步。一个完整周期有 28 步，并通过 9 个清楚的形态，从种子入土、裂壳、发芽一路长成结出金黄果实的大树，然后继续培育下一棵。

成长花园记录的是累积，不要求绝对连续。漏掉一天不会枯萎，也不会触发惩罚动画。

### 🏅 有温度的奖励，而不是交易

奖励系统由三部分组成：

| 方式 | 如何获得 | 想传达的意义 |
| --- | --- | --- |
| 系统勋章 | 达到清楚、可解释的成长里程碑 | 记住第一次、坚持、探索与重新出发 |
| 特别勋章 | 家长为具体成长事件手动颁发 | 留下家长真正观察到的进步 |
| 周心愿 | 孩子自主选择，一周内“今日达成”5 天后解锁 | 让孩子保有选择感，而不是用积分购买奖励 |

周心愿按周一至周日统计，不要求连续达成，每周最多解锁一个。没有达成时，心愿会温和地延续到下周，孩子也可以主动换选。

### ✨ 看见自己的成长轨迹

孩子可以在成长页查看：

- 365 天成长星图。
- 月度六领域视图，包括学习、阅读、运动、生活自理、家庭责任和兴趣探索。
- 每一天的任务、完成量、心情、自评、骄傲时刻与家长鼓励。
- 本周亮点、累计达成和成长花园进度。

家长可以查看趋势、领域分布、日期与领域矩阵及周期对比。所有比较都只面向孩子自己的过去，不做同龄排名。

### 👨‍👩‍👦 家长提供支持，不代替孩子打卡

家长中心可以：

- 创建、编辑、暂停或停用长期任务模板。
- 设置任务提醒、每日开始提醒和晚间回顾提醒。
- 回看并修正历史记录，补充家长鼓励。
- 创建心愿、颁发特别勋章并确认奖励领取。
- 查看成长分析，导出或恢复完整数据。

## 数据与隐私

- 无需注册账号，也不依赖服务端或 CloudKit。
- 任务、回顾、勋章和心愿通过 SwiftData 保存在设备本地。
- 家长可以导出版本化的明文 JSON 完整备份。
- 支持导出任务明细、每日汇总、量化记录和勋章记录 CSV。
- 文件可通过系统文件选择器保存到“文件”或 iCloud Drive，这不是自动云同步。
- 导入前会先验证并预览内容，默认按稳定标识合并，不静默覆盖已有记录。

> 导出的 JSON 和 CSV 是明文文件，可能包含孩子的成长记录，请妥善保管。

## 平台与辅助功能

- iOS / iPadOS 17 及以上版本。
- 当前 App 界面为中文。
- iPad 使用侧边栏布局，iPhone 使用底部标签栏，并适配横竖屏和分屏。
- 支持 Dynamic Type、VoiceOver、提高对比度和减少动态效果。
- 儿童常用操作保持较大的触控区域，并为打卡提供触觉、视觉和声音反馈。

## 开始运行

环境要求：Xcode 16.4 或更新版本。

```bash
git clone git@github.com:goby-ao/better-guozai.git
cd better-guozai
open GuozaiDay/GuozaiDay.xcodeproj
```

在 Xcode 中选择 `GuozaiDay` Scheme 和任意 iOS 17+ 的 iPad 或 iPhone，即可构建运行。首次启动会在设备本地创建“果仔”档案和一组可编辑的示例任务。

运行领域与数据测试：

```bash
cd GuozaiDay
swift test
```
